ClassBuilder(InstanceBuilder, Parentopt, Constructor, props) → {function}

Main class builder. It takes the constructor function and wraps it to add few automated processes for constructing a new class.

Properties and features:

  • set the parent class
  • save the inheritance chain
  • define the classes/functions/objects that are going to be encapsulated into the resulting class. The last encapsulated object will have a precedence over the previous objects, even parent class. Only the own properties of the new class will have the highest precedence
  • the reference to the parent class is stored in 'parent' property
  • inherited methods are stored in class prototype object
  • store default class properties in '__default' object where the objects will be copied, not shared

ParentConstructorFirst()

Execute a parent's constructor before the current constructor. The behavior of this builder is similar to C++ - all parent constructors are executing first, starting from the root parent constructor and finishing with the constructor of a new class. Before the constructor's chain will be executed the new instance will be extended by "__defaults" properties.
